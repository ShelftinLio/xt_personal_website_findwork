# 刘小天个人AI助手 - SKILL文件

## 📦 这个目录包含什么？

这是刘小天个人介绍AI助手的所有必需文件，可以直接集成到你的个人网站中。

## 📂 文件结构

```
xiaotian-ai-skill/
├── SKILL.md              # SKILL核心配置（必需）
├── README.md             # 本文件（使用说明）
└── data/                 # 数据源（必需）
    ├── 刘小天个人介绍.md
    └── 小红书文章资料归档_数据表.csv
```

## 🚀 如何使用

### 方法1：直接复制（推荐）

1. **在VSCode中复制整个文件夹**
   - 右键点击 `xiaotian-ai-skill` 文件夹
   - 选择"复制"
   - 粘贴到你的个人网站项目中

2. **确保路径正确**
   - 保持 `SKILL.md` 和 `data/` 的相对位置不变

### 方法2：手动复制

如果只需要特定文件，参考下面的文件说明。
